## Overview

This repository contains AI-powered content analysis systems designed to provide comprehensive insights into written content.

## Components

### HIFE (Human Insight Fusion Engine)

An advanced AI system designed to convert large written artefacts (blogs, emails, essays, transcripts, etc.) into multidimensional, humane, growth-oriented insight reports.

**Key Features:**
- Multidimensional analysis across linguistics, emotion, personality, cognition, social dynamics, values, expertise, and growth patterns
- Human-centered approach with radical respect for dignity, nuance, and context
- Comprehensive reporting with confidence scores and limitations
- Safety-first design with bias scanning and ethical safeguards

**Analysis Modules:**
- **Linguistics (LING)**: Readability, lexical diversity, stylometry, cohesion
- **Emotion (EMO)**: Sentiment analysis, affect mapping, empathy cues
- **Personality (PSY)**: Big Five, MBTI, Enneagram, attachment styles
- **Cognition (COG)**: Analytical vs. intuitive preferences, systems thinking
- **Social Dynamics (SOC)**: Persuasion, conflict style, leadership indicators
- **Values & Worldview (VAL)**: Value hierarchy, moral foundations, cultural lens
- **Expertise & Knowledge (EXP)**: Domain topics, depth vs. breadth
- **Growth Trajectory (GTH)**: Self-reflection, goal orientation, resilience
- **Risk Flags (RSK)**: Burnout, stress, ethical concerns

### ContentAnalyzer

An expert content analysis system utilizing advanced linguistic theories and evaluation frameworks.

**Evaluation Criteria:**
- Tone and Voice
- Language Usage
- Structure and Organization
- Audience Alignment
- Clarity and Coherence
- Persuasiveness
- Engagement
- Originality and Creativity
- Style Compliance
- Accuracy and Reliability
- Overall Effectiveness

**Advanced Analytical Methods:**
- Readability Analysis (Flesch-Kincaid, Gunning Fog)
- Sentiment Analysis
- Keyword Density and TF-IDF scoring
- Transition Word Usage Assessment
- Active vs. Passive Voice Ratio
- Paragraph and Sentence Length Variation
- Clich√© and Redundancy Detection
- Figurative Language Evaluation
- Jargon Appropriateness Check

## Usage

### HIFE
The HIFE system is configured via YAML and requires at least 500 words of user text for meaningful analysis. Configure the analysis by specifying:
- User context and analysis purpose
- Language preference
- Modules to focus on
- Output format (comprehensive, focused, or interactive)

### ContentAnalyzer
ContentAnalyzer provides detailed analysis reports with quality scores (0.0-1.0 scale) for each evaluation criterion, with specific evidence-based explanations.

## Philosophy

Both systems are designed with a focus on:
- Comprehensive and objective assessment
- Evidence-based evaluation
- Respect for human nuance and context
- Actionable insights and recommendations
- Transparency in confidence and limitations
